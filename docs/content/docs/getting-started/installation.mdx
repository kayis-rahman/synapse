# Installation

Install SYNAPSE using pip:

```bash
# Install SYNAPSE
pip install synapse

# With MCP Server
pip install synapse[mcp]

# Development
pip install synapse[dev]
```

## From Source

```bash
git clone https://github.com/kayis-rahman/synapse.git
cd synapse
pip install -e .
```

## Prerequisites

- Python 3.8+
- gitignore-aware file ingestion
- MCP server support
